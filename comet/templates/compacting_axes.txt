## Axis catalog (score only what you can judge from the turn; omit the rest)

**Skill** layer — about loaded skills + skill gaps:
- `gap_pattern` (0..1) — the trajectory needed a skill no existing skill covers. 1.0 = clear gap, candidate for a new skill.
- `clarity` (0..1) — the loaded skill's body was clear and actionable. Low = skill prose left the assistant guessing.
- `cross_project_applicability` (0..1) — what the turn taught generalises beyond this project.
- `task_completion_persistence` (0..1) — the turn evidence is about persistence past weak "done" signals.
- `comet_idiomatic_usage` (0..1) — the turn respects session-memory-first (vs treating memory as a search index).

**Tool** layer — about tool docstrings + per-turn ROI:
- `description_coverage` (0..1) — the relevant tool's docstring covers the use case the turn implies.
- `turn_tool_roi` (0..1) — THIS turn's tool use produced progress proportional to its cost. 1.0 = each call moved the deliverable; 0.0 = many calls without proportional progress (over-calling, redundant reads, looping). Score from the turn's call density vs visible progress.

**Prompt** layer — about harness section drift:
- `assistant_antipattern_match` (0..1) — the assistant turn matches a known anti-pattern a harness section should have prevented.
- `invariant_alignment` (0..1) — the section guidance the turn relies on stays consistent with framework invariants. Low = leakage or contradiction.

**MCP** layer — about external capability gaps:
- `mcp_gap_intent` (0..1) — the turn implies an external capability the agent lacks. Judge from concrete tool/service named + intent, NOT keyword match.

## Output

1. **evolve_axes**: dict of axis name → float in [0.0, 1.0]. Include only axes you can confidently judge from the turn. Missing keys are downstream-treated as no-signal — do not guess to fill the schema.
2. **evolve_reason**: one sentence, operator-facing, naming the failure mode or success pattern. Seeds the evolver's blame-attribution intent. Be concrete (e.g., "Assistant ran 18 read_file_tool calls returning >2KB each but produced no deliverable — over-calling pattern"). Avoid filler like "this turn warrants evolution."
